# 도커목록 (수업용 Docker 이미지 정리)

## 1) 화면 캡처 표준 (mcr.microsoft.com 사용)
- 원칙: 수업 자료용 UI/웹 화면 캡처가 필요한 경우 `mcr.microsoft.com` 계열 이미지를 사용합니다.
- 권장 이미지: `mcr.microsoft.com/playwright:v1.58.2-noble`
- 공식 참고: https://playwright.dev/docs/docker

### 캡처 예시 절차
1. 이미지 Pull
```bash
docker pull mcr.microsoft.com/playwright:v1.58.2-noble
```
2. 컨테이너 실행 후 캡처 스크립트 실행(프로젝트 마운트)
```bash
docker run --rm -it -v "$PWD":/work -w /work mcr.microsoft.com/playwright:v1.58.2-noble bash
# 컨테이너 내부에서
npm init -y
npm i playwright
node -e "const { chromium } = require('playwright'); (async()=>{const b=await chromium.launch({headless:true}); const p=await b.newPage(); await p.goto('http://127.0.0.1:8000'); await p.screenshot({path:'docs/screenshots/agent_ui.png', fullPage:true}); await b.close();})();"
```
